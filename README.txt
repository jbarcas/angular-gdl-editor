Installation instructions:

- Download and install NodeJS from http://nodejs.org.
- Download and install MongoDB from http://www.mongodb.org/downloads.
- In the root project directory (the one where package.json is), execute the following instruction on your command line: npm install. This will download some javascript modules for the application to run.
- Start your database server by executing: mongod (or <MONGODB_DIR>/bin/mongod).
- Start your web server by executing, in the root project directory: node app.js
- View in your browser at: http://localhost:8080
